### 指令的四种编码
- iABC -> A、B、C三个操作数，8、9、9位
- iABx -> A、Bx两个操作数，8、18位
- iAsBx -> A、sBx（有符号的整数），8、18位
- iAx -> Ax只携带一个操作数，占用26bit。
